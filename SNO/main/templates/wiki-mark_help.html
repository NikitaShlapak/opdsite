Доступные опции вики-разметки

Вики-разметка позволяет сделать текст логически структурированным и визульно привлекательным. В таблице представлены основные конструкции, примеры их использвания и аналоги на языке HTML.

# Описание Вики-разметка Отображение HTML
1 Курсив 
(( текст ))
 текст 
<i> текст </i>
2 Подчёркивание 
{{ текст }}
 текст 
<u> текст </u>
3 Жирное начертание 
[[ текст ]]
 текст 
<b> текст </b>
4 Горизонтальная разделительная линия --- 
<hr>
5 Заголовки 
#Большой заголовок
##Средний заголовок
###Маленький заголовок

Большой заголовок

Средний заголовок
Маленький заголовок
 
<h4>Большой заголовок</h4>
<h5>Средний заголовок</h5>
<h6>Маленький заголовок</h6>
4 Списки Нумерованный список 
<<olist>>
  <<newline>> Строка 1 <<endline>>
  <<newline>> Строка 2 <<endline>>
  <<newline>> Строка 3 <<endline>>
<<endlist>>
  1. Строка 1
  2. Строка 2
  3. Строка 3 
<ol>
  <li> Строка 1 </li>
  <li> Строка 2 </li>
  <li> Строка 3 </li>
</ol>
Маркированный список 
<<ulist>>
  <<newline>> Строка 1 <<endline>>
  <<newline>> Строка 2 <<endline>>
  <<newline>> Строка 3 <<endline>>
<<endlist>>
  • Строка 1
  • Строка 2
  • Строка 3 
<ul>
  <li> Строка 1 </li>
  <li> Строка 2 </li>
  <li> Строка 3 </li>
</ul>
